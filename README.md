##  Quick Start

### Prerequisites
- Java 21 or higher
- Maven 3.8+

### Installation

```bash
# Clone the repository
git clone https://github.com/AhmedQw1/SoundPlayer.git

# Navigate to project directory
cd SoundPlayer

# Build the project
mvn clean package

# Run the application
java -jar target/soundplayer-1.0.jar
